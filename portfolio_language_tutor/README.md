# LinguaLearn - Language Learning Web App

A professional, world-class single-page language learning application built with HTML, CSS, and JavaScript. Translate text, hear natural pronunciations, and practice with intelligent flashcards.

## Features

- **Instant Translation**: Translate text between multiple languages
- **Text-to-Speech**: Hear natural pronunciations using Web Speech API
- **Flashcards System**: Save and practice translations
- **Translation History**: Track your learning progress
- **Example Phrases**: Pre-built phrases for common situations
- **Offline Support**: Works without internet connection
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Accessibility**: Full keyboard navigation and screen reader support
- **Dark/Light Mode**: Automatic theme switching based on system preference

## Setup Instructions

### Quick Start

1. Clone or download the project files
2. Open `index.html` in a modern web browser
3. Start translating!
