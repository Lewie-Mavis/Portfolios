This folder contains a single-file Tailwind-powered site.

## Quick Start
1. Open `index.html` in a modern browser.
2. Edit contact details inside the file (search for `orders@oyatobeespoke.com` and Whatsapp number).
3. Host on Netlify/GitHub Pages or any static host.

## Sections
- Hero, About, Collections, Experience, Gallery, Measurements form, Contact.
- The form can:
  - Open your email client with a pre-filled message.
  - Open WhatsApp with a pre-filled message.
  - Print/Save the page as PDF.
